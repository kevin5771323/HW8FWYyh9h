# 项目简介

## 核心功能
- 支持图片、视频及摄像头进行玉米粒健康检测
- 提供9类玉米粒健康状态分类：破损、损伤、无机杂质、有机杂质、真菌感染、健康、未成熟、干瘪、虫蛀
- 支持选择检测模型和自定义置信度、IOU阈值
- 提供UI界面实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并直接查看所有检测结果
- 支持检测结果保存功能

## 适用场景
- 农产品质量检测中心
- 粮食加工厂质量控制
- 农业科研机构
- 玉米育种研究
- 农产品质量监管

## 技术特点
- 基于YOLOv8深度学习模型
- 使用PyQt5构建图形用户界面
- 兼容Python3.9环境
- 包含1200张标注数据集
- 提供完整源码和训练好的模型
